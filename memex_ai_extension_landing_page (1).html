How it works
Four steps, zero cloud

The entire pipeline runs locally. Ollama serves Llama 3.2 from your machine — your conversations are never uploaded anywhere.

01
🔍

Extract

Content script injects into the active tab and extracts the full message thread from the DOM.

02
✂️

Chunk

Long conversations are split into 4,000-character chunks to fit within the LLM's context window.

03
🧠

Summarize

Each chunk is sent to the background service worker, which calls Llama 3.2 via Ollama's local API.

04
📝

Combine

Per-chunk summaries are merged into one final summary and displayed in the popup — ready to copy or export.

// popup.js — chunked summarization
async function summarizeChat(text) {
  const chunks = chunkText(text, 4000);
  const summaries = [];

  for (const chunk of chunks) {
    const res = await callLlama(chunk);
    summaries.push(res);
  }

  // merge all chunk summaries
  return await callLlama(
    `Combine: ${summaries.join(" ")}`
  );
}

Get started in 5 minutes

You need Ollama running locally with Llama 3.2 pulled. Then load the extension — done.

1

Install Ollama and run ollama pull llama3.2 to download the model.

2

Set the env var: OLLAMA_ORIGINS=chrome-extension://* and restart Ollama.

3

Clone the repo, then load the folder as an unpacked extension in chrome://extensions.

4

Open any chat, click the LLM Bridge icon, hit Summarize — and watch the magic happen.
